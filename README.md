# CV project
Отзывчивая верстка макета резюме.

## Клонирование репозитория
```
$ git clone https://github.com/p-tka4enko/cv-project.git
```

## Сборка проекта
```
$ cd cv-project
$ npm i
$ npm run build
```

## Запуск сервера
```
$ npm run server
```
